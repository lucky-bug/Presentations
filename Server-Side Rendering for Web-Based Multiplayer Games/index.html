First: Event Collection

  • Collect events like mouse moves/clicks, key presses/releases, etc.
  • Prepare and send collected events to server.

Second: Event Handling

  • Update game/players based on the received events.
  • Render game on server.
  • Stream back rendered frames as video frames.

Third: Presentation

  • Design the view where game will be presented.
  • Embed receiving video frames in view.
